Rubik Documentation
===================

.. image:: images/rubik.png
   :scale: 40 %
   :align: right 

Rubik provides a simple and intuitive interface to create a wide variety of
mappings for structured communication patterns. Rubik supports a number of
elementary operations such as splits, tilts, or shifts, that can be combined
into a large number of unique patterns.
